#### 🧮 **반감기 계산 공식**

**월별 감소율**: 매월 5% (0.05) 고정 감소

```
현재_월_채굴률 = 기본_채굴률 × (1 - 월별_감소율 × 경과_개월수)
기본_채굴률 = 0.25 BW/시간
월별_감소율 = 0.05 (5%)
시작_월 = 2026년 6월
경과_개월수 = (현재_월 - 2026년 6월)
```

**구체적 계산 예시**:
- **2026년 6월**: 0.25 × (1 - 0.05 × 0) = **0.25 BW/시간** (0개월 경과, 0% 감소)
- **2026년 7월**: 0.25 × (1 - 0.05 × 1) = **0.2375 BW/시간** (1개월 경과, 5% 감소)
- **2026년 8월**: 0.25 × (1 - 0.05 × 2) = **0.225 BW/시간** (2개월 경과, 10% 감소)
- **2026년 12월**: 0.25 × (1 - 0.05 × 6) = **0.175 BW/시간** (6개월 경과, 30% 감소)
- **2027년 5월**: 0.25 × (1 - 0.05 × 11) = **0.1125 BW/시간** (11개월 경과, 55% 감소)

### 1.5. 오리지널 지갑 시스템 (Genesis Wallet System)

**BitWishNetwork**는 **BitWish 독자 개발 블록체인** 기반으로 **5개의 시스템 지갑**을 생성하여 초기 토큰을 분배하고 수수료를 관리합니다:

#### 🏦 **시스템 지갑 구성**

**[토큰 분배 지갑 3개] - 총 210억 BW**

| 지갑명 | 비율 | 수량 | 용도 |
|--------|------|------|------|
| **COMMUNITY_70PERCENT** | 70% | 14,700,000,000 BW | 회원 마이닝 배분 (생태계 80% 중 70%) |
| **PARTNER_10PERCENT** | 10% | 2,100,000,000 BW | 파트너/상점 배분 (생태계 80% 중 10%) |
| **FOUNDATION_20PERCENT** | 20% | 4,200,000,000 BW | 재단/개발팀/운영 |

**[수수료 지갑 2개] - 0 BW 시작**

| 지갑명 | 비율 | 초기 수량 | 용도 |
|--------|------|----------|------|
| **ECOSYSTEM_FEE_70** | 70% | 0 BW | 수수료의 70% 수집 (생태계 조성) |
| **DEV_FEE_30** | 30% | 0 BW | 수수료의 30% 수집 (개발팀 운영) |

#### 🔐 **시스템 지갑 특징**

- **생성 시점**: BitWish 블록체인 제네시스 블록 생성과 동시에 생성
- **토큰 분배**: 각 지갑에 지정된 수량의 BW 토큰이 즉시 입금
- **지갑 유형**: BitWish 독자 개발 블록체인 기반 고급 보안 지갑
- **접근 권한**: 각 카테고리별 관리자만 접근 가능
- **트랜잭션**: 블록체인 상에서 투명하게 기록 및 추적
- **실시간 조회**: MongoDB 기반으로 실시간 잔액 및 거래 내역 조회 가능

#### 💰 **BitWish 토큰 분배 및 수수료 메커니즘**

1. **BitWish 제네시스 블록 생성** 시 5개 시스템 지갑 자동 생성
2. **총 210억 BW** 토큰이 분배 지갑 3개에 비율에 맞춰 분배
3. **수수료 지갑 2개**는 0 BW로 시작하여 블록 생성 및 거래 수수료 자동 수집
4. **지갑별 잔액**은 BitWish 블록체인 상에서 실시간으로 확인 가능
5. **회원 보상 지급**은 COMMUNITY_70PERCENT 지갑에서 사용자 지갑으로 전송

### 1.6. 보상 시스템

**BitWishNetwork**는 **3가지 보너스 시스템**을 통해 사용자의 참여를 적극적으로 보상합니다:

#### 📅 **출석 보너스 (Attendance Bonus)**

**BitWishNetwork의 출석 보너스는 사용자의 꾸준한 참여를 보상하는 핵심 시스템입니다:**

##### 🎯 **출석 보너스 체계**

| 출석 상태 | 보너스율 | 설명 | 일일 보상 증가 |
|-----------|----------|------|----------------|
| **출석 체크 완료** | **5% (0.05)** | 🎯 **일일 출석 보너스** | **0.2625 BW/시간** |
| **출석 체크 미완료** | **0% (0.00)** | ❌ **출석 보너스 없음** | **0.25 BW/시간** |

**🎯 출석 보너스 시스템 특징:**
- **단순화된 구조**: 연속 출석일수와 관계없이 출석 체크 시 일괄 5% 보너스
- **시간 제한**: 마지막 출석일로부터 24시간 유예 기간 내에만 보너스 적용
- **일일 체크**: 매일 출석 체크를 완료하면 5% 보너스 획득
- **출석 체크 방법**: 
예1) 출석 가능 날짜: 정각 오전 9시 당일 달력 날짜는 활성화("파란색")
예2) 출석 체크 상실 날짜: 전날 1일 정각 오전 9시부터 다음 날 2일 오전 8시 59분 59초까지 달력 날짜는 활성화("파란색")로 유지 되지만 2일 정각 오전 9시가되면 1일 날짜는 즉시 비활성화("회색"처리되고 영원히 유지됨)처리된다.
예3) 출석 체크: 1일 정각 오전 9시 부터 ~ 2일 오전 8시 59분 59초전 까지 출석 체크를 했다면 1일 날짜는 비활성화("빨간색"처리되고 영원히 유지되며 보너스 5%가 즉시 적용되고 기본채굴률에 포함되어 실시간 총 채굴량에 누적 반영되고 2일 정각 오전 9시가되면 0%로 초기화 된다.)

**⚠️ 중요**: 출석 보너스는 마이닝 시작 후에만 받을 수 있습니다.

##### 🧮 **출석 보너스 계산 공식**
```
출석_보너스율 = 출석_체크_완료시_5% (0.05)
최종_채굴률 = 기본_채굴률 × (1 + 출석_보너스율)
```

##### 📊 **구체적 계산 예시**

**예시 1: 출석 체크 완료 사용자**
- **기본 채굴률**: 0.25 BW/시간
- **출석 보너스**: 5% (0.05)
- **최종 채굴률**: 0.25 × (1 + 0.05) = **0.2625 BW/시간**
- **일일 추가 수익**: 0.0125 × 24 = **0.3 BW/일**
- **월간 추가 수익**: 0.3 × 30 = **9 BW/월**

**예시 2: 출석 체크 미완료 사용자**
- **기본 채굴률**: 0.25 BW/시간
- **출석 보너스**: 0% (0.00)
- **최종 채굴률**: 0.25 × (1 + 0.00) = **0.25 BW/시간**

#### 👥 **추천인 보너스 (Referral Bonus)**

**BitWishNetwork의 추천인 보너스는 커뮤니티 확장을 위한 가장 강력한 동기부여 시스템입니다:**

##### 🎁 **추천인 특별 혜택**
- **즉시 보너스**: 추천인코드 발급자와 가입자(지갑생성자)는 각 **1 BW 즉시 지급** (첫 지갑 생성 시)
- **추천인 보너스율**: 발급자는 **2% 영구 적용** (가입자 발생 시 마다 추가 반영영), 가입자는 **가입 시 1회만 2% 영구 적용** (재적용 불가)
- **영구 관계**: 추천인 관계는 한 번 설정되면 영구적으로 유지
- **KYC 연동**: KYC 통과 시에만 보너스 지급으로 가짜 계정 방지
- **실시간 추적**: 추천인 수와 보너스율을 실시간으로 확인 가능

##### 🔒 **추천인 보너스 분리 시스템**

**가짜 계정 방지를 위한 혁신적인 시스템:**
1. **보너스 2% 풀 분리**: 추천인 보너스 2%를 별도 풀에 안전하게 실시간 채굴 반영되어 실시간 보관
2. **보너스 1BW 분리**: 추천인 보상 1BW 별도 분리 보관
3. **KYC 연동**: KYC 통과 시에만 보상 1BW, 보너스 2% 실시간 보관 됐던 BW지급
4. **가짜 계정 필터링**: KYC 미통과 계정은 보너스 지급 불가

- **계산 정확성 보장**: 모든 보상 부동소수점 정밀 계산에서 수학적 완벽성 달성
- **사용자 신뢰성**: 정확한 수치로 투명하고 신뢰할 수 있는 보상 시스템

## 3. 기술 아키텍처

### 3.1. BitWish 독자 개발 블록체인

**BitWishNetwork**는 **완전한 독자 개발 블록체인**으로 구축되었으며, **고급 합의 프로토콜**과 **PoW(Proof of Work)**를 결합하여 높은 처리량과 낮은 수수료를 제공합니다.

#### 🎯 **고급 정밀 계산 아키텍처**

**BitWishNetwork의 핵심 기술적 우위는 금융권 표준을 초월하는 정밀 계산 시스템입니다:**

**🔬 Decimal.js 고급 설정 (업계 최고 수준)**
```typescript
Decimal.set({
  precision: 50,        // 최대 정밀도 50자리 (업계 최고)
  rounding: Decimal.ROUND_HALF_UP,  // 금융권 표준 반올림
  toExpNeg: -50,        // 음수 지수 범위
  toExpPos: 50,         // 양수 지수 범위
  maxE: 9e15,           // 최대 지수
  minE: -9e15,          // 최소 지수
  modulo: Decimal.ROUND_FLOOR,  // 나머지 연산 방식
  crypto: false         // 암호화 모드 비활성화
});
```

**💎 금융권 표준 완전 준수**
- **ISO 4217 국제 통화 표준** 준수
- **BigDecimal 기반 정밀 연산** (금융권 표준 방식)
- **부동소수점 오차 완전 제거** (0.1 + 0.2 = 0.3 정확)
- **실시간 정밀도 검증** 시스템 구축

**🚀 블록체인 시장 대비 기술적 우위**
- **Bitcoin**: 8자리 정밀도 (정수 연산 한계) → **BitWish**: 50자리 정밀도 (소수점 정밀 계산)
- **Ethereum**: 18자리 정밀도 → **BitWish**: 50자리 정밀도 (2.8배 우수)
- **Stellar**: 7자리 정밀도 → **BitWish**: 50자리 정밀도 (7.1배 우수)

#### 🔗 **BitWish 독자 개발 블록체인 아키텍처**

```
[사용자 브라우저] ←→ [React 프론트엔드:4000]
         ↓
```
