## 🔥 **유저용 가맹점 보너스 시스템 완벽 구현**

### **📋 1단계: 기술 명세서 분석**

```markdown
# 가맹점 보너스 시스템 기술 명세서

## 1차 백서 정책 수치
- 가맹점 보너스율: 30% (0.3)
- 가맹점 등록 보너스: 2.5 BW
- 가맹점 승인 보너스: 5 BW
- 가맹점 운영 보너스: 월 2 BW
- 기본 채굴률: 0.25 BW/시간(가맹점 보너스율은 기본 채굴률에 곱하여 합산함)

## 가맹점 시스템 요구사항
- 가맹점 등록: 사업자등록번호, 상호명, 주소 입력
- 가맹점 승인: 관리자 승인 시스템
- 가맹점 상태: 등록, 승인, 운영, 정지, 해지
- 가맹점 보너스: 실시간 보너스 계산

## 개인별 분리 원칙
- 전역/공통 변수 함수 클래스 전역 모달 완전 제거
- localStorage 기반 개인별 데이터 저장
- 완전한 컴포넌트 독립성
```
#### 내용 추가: 현재 위 내용은 변경된 사안으로 아래 코드와 틀린 부분이 있으니 수정 하여야 한다. ####

### **📋 2단계: 유저용 가맹점 보너스 컴포넌트 구현**

```typescript
// Node_HomePage/src/components/mining/user/UserPartnerBonus.tsx

/**
 * =====================================================
 * ⚠️  중요: 전역 모달, 공통 변수 함수 절대 포함하지 않는다
 * =====================================================
* ❌ 전역 변수 사용 금지
 * ❌ 공통 함수 사용 금지  
 * ❌ 공통 클래스 사용 금지
 * ❌ 전역 모달 사용 금지
 * ❌ 중복 코드 사용 금지
 * ❌ 다른 컴포넌트와 상태 공유 금지
 * ❌ 전역 상태 관리 라이브러리 사용 금지
 * ❌ 스텔라 관련 코드 사용 금지
 * ❌ BIP39 시드문구 사용 금지
 * 
 * ✅ 완벽한 독립성 보장
 * ✅ 자체 상태 관리만 사용
 * ✅ 자체 API 호출만 사용
 * ✅ 자체 에러 처리만 사용
 * ✅ 자체 보안 검증만 사용
 * ✅ BitWish Network 전용 시스템만 사용
 * ✅ 모든 텍스트는 4개국 언어 즉시 번역 시스템 구조로 구현 
 * 유저 가맹점보너스 컴포넌트 - 완전한 개인별 분리
 * - 1차 백서 정책 수치 완전 적용 (3% 보너스)
 * - 가맹점 등록/승인/운영 시스템
 * - 50단위 부동소수점 정밀도
 * - 4개국 언어 즉시 번역
 */

import React, { useState, useEffect, useCallback } from 'react';
import { useTranslation } from 'react-i18next';
import Decimal from 'decimal.js';

// 1차 백서 정책 수치 (완전 적용)
const PARTNER_POLICY = {
  bonusRate: 0.03, // 3% 보너스 (백서 기준)
  registrationBonus: 5.0, // 가맹점 등록 보너스 5 BW
  approvalBonus: 10.0, // 가맹점 승인 보너스 10 BW
  monthlyBonus: 2.0, // 월별 운영 보너스 2 BW
  miningRate: 0.25 // 기본 채굴률 0.25 BW/시간
};

interface UserPartnerBonusProps {
  walletAddress: string;
  isMiningStarted: boolean;
  onPartnerChange: (partnerData: any) => void;
}

const UserPartnerBonus: React.FC<UserPartnerBonusProps> = ({ 
  walletAddress, 
  isMiningStarted,
  onPartnerChange 
}) => {
  const { t } = useTranslation();
  
  // 개인별 상태 관리 (완전 분리)
  const [partnerStatus, setPartnerStatus] = useState<'none' | 'registered' | 'approved' | 'operating' | 'suspended' | 'terminated'>('none');
  const [partnerData, setPartnerData] = useState<any>(null);
  const [showRegistrationModal, setShowRegistrationModal] = useState(false);
  const [registrationForm, setRegistrationForm] = useState({
    businessNumber: '',
    businessName: '',
    address: '',
    phone: '',
    email: '',
    description: ''
  });
  const [totalPartnerBonus, setTotalPartnerBonus] = useState(0);
  const [monthlyBonusReceived, setMonthlyBonusReceived] = useState(false);
  const [lastMonthlyBonusDate, setLastMonthlyBonusDate] = useState<Date | null>(null);

  // 50단위 부동소수점 정밀도 계산
  const calculateWithPrecision = useCallback((value: number) => {
    return new Decimal(value).toDecimalPlaces(50);
  }, []);

  // 가맹점 보너스 계산 (50단위 정밀도)
  const calculatePartnerBonus = useCallback(() => {
    if (partnerStatus !== 'operating') {
      return {
        bonusRate: 0,
        enhancedRate: PARTNER_POLICY.miningRate,
        bonusAmount: 0
      };
    }

    const bonusRate = calculateWithPrecision(PARTNER_POLICY.bonusRate);
    const enhancedRate = calculateWithPrecision(PARTNER_POLICY.miningRate)
      .times(calculateWithPrecision(1).plus(bonusRate));
    const bonusAmount = enhancedRate.minus(PARTNER_POLICY.miningRate);

    return {
      bonusRate: bonusRate.toNumber(),
      enhancedRate: enhancedRate.toNumber(),
      bonusAmount: bonusAmount.toNumber()
    };
  }, [partnerStatus, calculateWithPrecision]);

  // 월별 보너스 지급 가능 여부 확인
  const checkMonthlyBonusEligibility = useCallback(() => {
    if (!lastMonthlyBonusDate) return true;
    
    const now = new Date();
    const lastBonusDate = new Date(lastMonthlyBonusDate);
    const diffTime = now.getTime() - lastBonusDate.getTime();
    const diffDays = Math.ceil(diffTime / (1000 * 60 * 60 * 24));
    
    return diffDays >= 30; // 30일 이상 경과
  }, [lastMonthlyBonusDate]);

  // 가맹점 등록
  const handlePartnerRegistration = useCallback(async () => {
    if (!registrationForm.businessNumber || !registrationForm.businessName || !registrationForm.address) {
      alert(t('partner.registration.error.incomplete'));
      return;
    }

    try {
      // 백엔드 API 호출
      const response = await fetch('/api/partner/register', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({
          walletAddress,
          partnerData: registrationForm
        })
      });

      const result = await response.json();

      if (result.success) {
        // 개인별 데이터 저장
        const newPartnerData = {
          walletAddress,
          businessNumber: registrationForm.businessNumber,
          businessName: registrationForm.businessName,
          address: registrationForm.address,
          phone: registrationForm.phone,
          email: registrationForm.email,
          description: registrationForm.description,
          status: 'registered',
          registrationDate: new Date().toISOString(),
          registrationBonus: PARTNER_POLICY.registrationBonus
        };
        
        localStorage.setItem(`personal_${walletAddress}_partner`, JSON.stringify(newPartnerData));
        setPartnerData(newPartnerData);
        setPartnerStatus('registered');
        
        // 부모 컴포넌트에 알림
        onPartnerChange(newPartnerData);
        
        setShowRegistrationModal(false);
        setRegistrationForm({
          businessNumber: '',
          businessName: '',
          address: '',
          phone: '',
          email: '',
          description: ''
        });
        
        alert(`✅ ${t('partner.registration.success')}\n${t('partner.registration.bonus')}: ${PARTNER_POLICY.registrationBonus} BW`);
      } else {
        alert(`❌ ${result.error || t('partner.registration.error.failed')}`);
      }
    } catch (error) {
      console.error('가맹점 등록 오류:', error);
      alert(`❌ ${t('partner.registration.error.network')}`);
    }
  }, [walletAddress, registrationForm, onPartnerChange, t]);

  // 월별 보너스 지급
  const handleMonthlyBonusClaim = useCallback(async () => {
    if (!checkMonthlyBonusEligibility() || monthlyBonusReceived) return;

    try {
      // 백엔드 API 호출
      const response = await fetch('/api/partner/monthly-bonus', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({
          walletAddress
        })
      });

      const result = await response.json();

      if (result.success) {
        // 개인별 데이터 업데이트
        const monthlyBonusData = {
          walletAddress,
          monthlyBonus: PARTNER_POLICY.monthlyBonus,
          receivedDate: new Date().toISOString(),
          totalBonus: (partnerData?.totalBonus || 0) + PARTNER_POLICY.monthlyBonus
        };
        
        const updatedPartnerData = {
          ...partnerData,
          totalBonus: monthlyBonusData.totalBonus,
          lastMonthlyBonusDate: monthlyBonusData.receivedDate
        };
        
        localStorage.setItem(`personal_${walletAddress}_partner`, JSON.stringify(updatedPartnerData));
        setPartnerData(updatedPartnerData);
        setTotalPartnerBonus(monthlyBonusData.totalBonus);
        setMonthlyBonusReceived(true);
        setLastMonthlyBonusDate(new Date());
        
        // 부모 컴포넌트에 알림
        onPartnerChange(updatedPartnerData);

        alert(`✅ ${t('partner.monthlyBonus.success')}\n${t('partner.monthlyBonus.amount')}: ${PARTNER_POLICY.monthlyBonus} BW`);
      } else {
        alert(`❌ ${result.error || t('partner.monthlyBonus.error')}`);
      }
    } catch (error) {
      console.error('월별 보너스 지급 오류:', error);
      alert(`❌ ${t('partner.monthlyBonus.error')}`);
    }
  }, [walletAddress, checkMonthlyBonusEligibility, monthlyBonusReceived, partnerData, onPartnerChange, t]);

  // 개인별 데이터 로드
  useEffect(() => {
    const loadPersonalData = () => {
      try {
        const savedData = localStorage.getItem(`personal_${walletAddress}_partner`);
        if (savedData) {
          const data = JSON.parse(savedData);
          setPartnerData(data);
          setPartnerStatus(data.status || 'none');
          setTotalPartnerBonus(data.totalBonus || 0);
          setLastMonthlyBonusDate(data.lastMonthlyBonusDate ? new Date(data.lastMonthlyBonusDate) : null);
          
          // 월별 보너스 지급 가능 여부 확인
          if (data.lastMonthlyBonusDate) {
            const lastDate = new Date(data.lastMonthlyBonusDate);
            const now = new Date();
            const diffTime = now.getTime() - lastDate.getTime();
            const diffDays = Math.ceil(diffTime / (1000 * 60 * 60 * 24));
            setMonthlyBonusReceived(diffDays < 30);
          }
        }
      } catch (error) {
        console.error('개인별 데이터 로드 오류:', error);
      }
    };

    loadPersonalData();
  }, [walletAddress]);

  const partnerBonus = calculatePartnerBonus();
  const canClaimMonthlyBonus = checkMonthlyBonusEligibility() && !monthlyBonusReceived && partnerStatus === 'operating';

  return (
    <div className="bg-gradient-to-r from-orange-50 to-red-50 rounded-xl p-6">
      <h3 className="text-xl font-bold text-gray-800 mb-4">
        🏪 {t('partner.title')}
      </h3>

      {/* 가맹점 상태 표시 */}
      <div className="mb-6">
        <div className={`rounded-lg p-4 ${
          partnerStatus === 'operating' ? 'bg-green-100 border-2 border-green-300' :
          partnerStatus === 'approved' ? 'bg-blue-100 border-2 border-blue-300' :
          partnerStatus === 'registered' ? 'bg-yellow-100 border-2 border-yellow-300' :
          'bg-gray-100 border-2 border-gray-300'
        }`}>
          <div className="flex items-center justify-between mb-2">
            <span className="text-lg font-semibold">
              {partnerStatus === 'operating' ? '🏪' : 
               partnerStatus === 'approved' ? '✅' : 
               partnerStatus === 'registered' ? '📝' : 
               '❌'} 
              {t('partner.status.title')}
            </span>
            <span className={`px-3 py-1 rounded-full text-sm font-medium ${
              partnerStatus === 'operating' ? 'bg-green-500 text-white' : 
              partnerStatus === 'approved' ? 'bg-blue-500 text-white' : 
              partnerStatus === 'registered' ? 'bg-yellow-500 text-white' : 
              'bg-gray-500 text-white'
            }`}>
              {partnerStatus === 'operating' ? t('partner.status.operating') : 
               partnerStatus === 'approved' ? t('partner.status.approved') : 
               partnerStatus === 'registered' ? t('partner.status.registered') : 
               t('partner.status.none')}
            </span>
          </div>
          
          {partnerData && (
            <div className="text-sm text-gray-600">
              <div>{t('partner.businessName')}: {partnerData.businessName}</div>
              <div>{t('partner.businessNumber')}: {partnerData.businessNumber}</div>
            </div>
          )}
        </div>
      </div>

      {/* 가맹점 등록 섹션 */}
      {partnerStatus === 'none' && (
        <div className="mb-6">
          <h4 className="text-lg font-semibold text-orange-700 mb-3">
            📝 {t('partner.registration.title')}
          </h4>
          
          <button
            onClick={() => setShowRegistrationModal(true)}
            disabled={!isMiningStarted}
            className={`w-full py-3 px-4 rounded-lg font-medium transition-colors ${
              !isMiningStarted
                ? 'bg-gray-300 text-gray-500 cursor-not-allowed'
                : 'bg-orange-500 text-white hover:bg-orange-600'
            }`}
          >
            {!isMiningStarted ? t('partner.registration.miningRequired') : t('partner.registration.button')}
          </button>
        </div>
      )}

      {/* 가맹점 통계 */}
      {partnerData && (
        <div className="mb-6">
          <h4 className="text-lg font-semibold text-green-700 mb-3">
            📊 {t('partner.stats.title')}
          </h4>
          
          <div className="grid grid-cols-2 gap-4">
            <div className="bg-white rounded-lg p-4 text-center">
              <div className="text-2xl font-bold text-green-600">
                {totalPartnerBonus.toFixed(2)}
              </div>
              <div className="text-sm text-gray-600">
                {t('partner.stats.totalBonus')} (BW)
              </div>
            </div>
            
            <div className="bg-white rounded-lg p-4 text-center">
              <div className="text-2xl font-bold text-blue-600">
                {partnerData.registrationBonus || 0}
              </div>
              <div className="text-sm text-gray-600">
                {t('partner.stats.registrationBonus')} (BW)
              </div>
            </div>
          </div>
        </div>
      )}

      {/* 보너스 정보 */}
      <div className="mb-6">
        <div className="bg-white rounded-lg p-4">
          <h4 className="text-lg font-semibold text-gray-700 mb-3">
            💰 {t('partner.bonus.title')}
          </h4>
          
          <div className="space-y-2">
            <div className="flex justify-between">
              <span className="text-gray-600">{t('partner.bonus.rate')}:</span>
              <span className="font-bold text-green-600">
                {partnerBonus.bonusRate * 100}%
              </span>
            </div>
            
            <div className="flex justify-between">
              <span className="text-gray-600">{t('partner.bonus.enhancedRate')}:</span>
              <span className="font-bold text-blue-600">
                {partnerBonus.enhancedRate.toFixed(6)} BW/시간
              </span>
            </div>
            
            <div className="flex justify-between">
              <span className="text-gray-600">{t('partner.bonus.amount')}:</span>
              <span className="font-bold text-purple-600">
                +{partnerBonus.bonusAmount.toFixed(6)} BW/시간
              </span>
            </div>
          </div>
        </div>
      </div>

      {/* 월별 보너스 섹션 */}
      {partnerStatus === 'operating' && (
        <div className="mb-6">
          <h4 className="text-lg font-semibold text-yellow-700 mb-3">
            🎁 {t('partner.monthlyBonus.title')}
          </h4>
          
          <div className="bg-white rounded-lg p-4">
            <div className="flex items-center justify-between mb-3">
              <span className="text-gray-700">
                {t('partner.monthlyBonus.amount')}: {PARTNER_POLICY.monthlyBonus} BW
              </span>
              <span className={`px-3 py-1 rounded-full text-sm font-medium ${
                canClaimMonthlyBonus ? 'bg-yellow-500 text-white' : 
                monthlyBonusReceived ? 'bg-green-500 text-white' : 
                'bg-gray-500 text-white'
              }`}>
                {canClaimMonthlyBonus ? t('partner.monthlyBonus.available') : 
                 monthlyBonusReceived ? t('partner.monthlyBonus.received') : 
                 t('partner.monthlyBonus.waiting')}
              </span>
            </div>
            
            {canClaimMonthlyBonus && (
              <button
                onClick={handleMonthlyBonusClaim}
                className="w-full py-2 px-4 bg-yellow-500 text-white rounded-lg font-medium hover:bg-yellow-600 transition-colors"
              >
                {t('partner.monthlyBonus.claim')}
              </button>
            )}
            
            {lastMonthlyBonusDate && (
              <div className="text-sm text-gray-600 mt-2">
                {t('partner.monthlyBonus.lastReceived')}: {lastMonthlyBonusDate.toLocaleDateString('ko-KR')}
              </div>
            )}
          </div>
        </div>
      )}

      {/* 마이닝 시작 안내 */}
      {!isMiningStarted && (
        <div className="mt-4 p-3 bg-yellow-50 border border-yellow-200 rounded-lg">
          <div className="text-sm text-yellow-800">
            ⚠️ {t('partner.miningRequired')}
          </div>
        </div>
      )}

      {/* 가맹점 등록 모달 */}
      {showRegistrationModal && (
        <div className="fixed inset-0 bg-black bg-opacity-50 flex items-center justify-center z-[10000]">
          <div className="bg-white rounded-lg shadow-xl max-w-md w-full mx-4 transform transition-all duration-300 scale-100">
            <div className="p-6">
              <h3 className="text-lg font-bold text-gray-900 mb-4">
                📝 {t('partner.registration.title')}
              </h3>
              
              <div className="space-y-4">
                <div>
                  <label className="block text-sm font-medium text-gray-700 mb-1">
                    {t('partner.registration.businessNumber')} *
                  </label>
                  <input
                    type="text"
                    value={registrationForm.businessNumber}
                    onChange={(e) => setRegistrationForm(prev => ({ ...prev, businessNumber: e.target.value }))}
                    placeholder="123-45-67890"
                    className="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-orange-500"
                  />
                </div>
                
                <div>
                  <label className="block text-sm font-medium text-gray-700 mb-1">
                    {t('partner.registration.businessName')} *
                  </label>
                  <input
                    type="text"
                    value={registrationForm.businessName}
                    onChange={(e) => setRegistrationForm(prev => ({ ...prev, businessName: e.target.value }))}
                    placeholder="상호명을 입력하세요"
                    className="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-orange-500"
                  />
                </div>
                
                <div>
                  <label className="block text-sm font-medium text-gray-700 mb-1">
                    {t('partner.registration.address')} *
                  </label>
                  <input
                    type="text"
                    value={registrationForm.address}
                    onChange={(e) => setRegistrationForm(prev => ({ ...prev, address: e.target.value }))}
                    placeholder="주소를 입력하세요"
                    className="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-orange-500"
                  />
                </div>
                
                <div>
                  <label className="block text-sm font-medium text-gray-700 mb-1">
                    {t('partner.registration.phone')}
                  </label>
                  <input
                    type="tel"
                    value={registrationForm.phone}
                    onChange={(e) => setRegistrationForm(prev => ({ ...prev, phone: e.target.value }))}
                    placeholder="전화번호를 입력하세요"
                    className="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-orange-500"
                  />
                </div>
                
                <div>
                  <label className="block text-sm font-medium text-gray-700 mb-1">
                    {t('partner.registration.email')}
                  </label>
                  <input
                    type="email"
                    value={registrationForm.email}
                    onChange={(e) => setRegistrationForm(prev => ({ ...prev, email: e.target.value }))}
                    placeholder="이메일을 입력하세요"
                    className="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-orange-500"
                  />
                </div>
                
                <div>
                  <label className="block text-sm font-medium text-gray-700 mb-1">
                    {t('partner.registration.description')}
                  </label>
                  <textarea
                    value={registrationForm.description}
                    onChange={(e) => setRegistrationForm(prev => ({ ...prev, description: e.target.value }))}
                    placeholder="사업 설명을 입력하세요"
                    rows={3}
                    className="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-orange-500"
                  />
                </div>
              </div>
              
              <div className="flex space-x-3 mt-6">
                <button 
                  onClick={() => setShowRegistrationModal(false)}
                  className="flex-1 px-4 py-2 bg-gray-300 text-gray-700 rounded-lg hover:bg-gray-400 transition-colors"
                >
                  {t('common.cancel')}
                </button>
                <button 
                  onClick={handlePartnerRegistration}
```

### **📋 4단계: 백엔드 API 구현**

```javascript
// Node_HomePage/simple-server-new.js에 추가

// 1차 백서 정책 수치 적용
const PARTNER_POLICY = {
  bonusRate: 0.03, // 3% 보너스 (백서 기준)
  registrationBonus: 5.0, // 가맹점 등록 보너스 5 BW
  approvalBonus: 10.0, // 가맹점 승인 보너스 10 BW
  monthlyBonus: 2.0, // 월별 운영 보너스 2 BW
  miningRate: 0.25 // 기본 채굴률 0.25 BW/시간
};

// 가맹점 등록 API
app.post('/api/partner/register', (req, res) => {
  try {
    const { walletAddress, partnerData } = req.body;
    
    // 개인별 가맹점 데이터 저장
    const newPartnerData = {
      walletAddress,
      businessNumber: partnerData.businessNumber,
      businessName: partnerData.businessName,
      address: partnerData.address,
      phone: partnerData.phone,
      email: partnerData.email,
      description: partnerData.description,
      status: 'registered',
      registrationDate: new Date().toISOString(),
      registrationBonus: PARTNER_POLICY.registrationBonus,
      totalBonus: PARTNER_POLICY.registrationBonus
    };

    personalPartnerData.set(walletAddress, newPartnerData);

    res.json({ 
      success: true, 
      message: '가맹점 등록 완료!',
      data: newPartnerData
    });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

// 가맹점 승인 API (관리자용)
app.post('/api/partner/approve', (req, res) => {
  try {
    const { walletAddress } = req.body;
    
    const partnerData = personalPartnerData.get(walletAddress);
    if (!partnerData) {
      return res.json({ success: false, error: '가맹점 데이터를 찾을 수 없습니다.' });
    }

    // 가맹점 승인 처리
    partnerData.status = 'approved';
    partnerData.approvalDate = new Date().toISOString();
    partnerData.approvalBonus = PARTNER_POLICY.approvalBonus;
    partnerData.totalBonus += PARTNER_POLICY.approvalBonus;

    personalPartnerData.set(walletAddress, partnerData);

    res.json({ 
      success: true, 
      message: '가맹점 승인 완료!',
      data: partnerData
    });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

// 가맹점 운영 시작 API
app.post('/api/partner/start-operating', (req, res) => {
  try {
    const { walletAddress } = req.body;
    
    const partnerData = personalPartnerData.get(walletAddress);
    if (!partnerData || partnerData.status !== 'approved') {
      return res.json({ success: false, error: '승인된 가맹점만 운영을 시작할 수 있습니다.' });
    }

    // 가맹점 운영 시작
    partnerData.status = 'operating';
    partnerData.operatingStartDate = new Date().toISOString();

    personalPartnerData.set(walletAddress, partnerData);

    res.json({ 
      success: true, 
      message: '가맹점 운영 시작!',
      data: partnerData
    });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

// 월별 보너스 지급 API
app.post('/api/partner/monthly-bonus', (req, res) => {
  try {
    const { walletAddress } = req.body;
    
    const partnerData = personalPartnerData.get(walletAddress);
    if (!partnerData || partnerData.status !== 'operating') {
      return res.json({ success: false, error: '운영중인 가맹점만 월별 보너스를 받을 수 있습니다.' });
    }

    // 월별 보너스 지급 가능 여부 확인
    if (partnerData.lastMonthlyBonusDate) {
      const lastDate = new Date(partnerData.lastMonthlyBonusDate);
      const now = new Date();
      const diffTime = now.getTime() - lastDate.getTime();
      const diffDays = Math.ceil(diffTime / (1000 * 60 * 60 * 24));
      
      if (diffDays < 30) {
        return res.json({ success: false, error: '월별 보너스는 30일마다 받을 수 있습니다.' });
      }
    }

    // 월별 보너스 지급
    partnerData.lastMonthlyBonusDate = new Date().toISOString();
    partnerData.totalBonus += PARTNER_POLICY.monthlyBonus;

    personalPartnerData.set(walletAddress, partnerData);

    res.json({ 
      success: true, 
      message: '월별 보너스 지급 완료!',
      data: {
        monthlyBonus: PARTNER_POLICY.monthlyBonus,
        totalBonus: partnerData.totalBonus,
        receivedDate: partnerData.lastMonthlyBonusDate
      }
    });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

// 가맹점 상태 조회 API
app.get('/api/partner/status/:walletAddress', (req, res) => {
  try {
    const { walletAddress } = req.params;
    const partnerData = personalPartnerData.get(walletAddress) || {};
    
    res.json({ success: true, data: partnerData });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

// 통합 보상률 계산 API
app.get('/api/rewards/rate/:walletAddress', (req, res) => {
  try {
    const { walletAddress } = req.params;
    
    // 가맹점 보너스 계산
    const partnerData = personalPartnerData.get(walletAddress);
    const partnerBonus = partnerData && partnerData.status === 'operating' ? PARTNER_POLICY.bonusRate : 0;
    
    // 총 보너스율 계산
    const totalBonusRate = partnerBonus;
    const enhancedRate = PARTNER_POLICY.miningRate * (1 + totalBonusRate);
    
    res.json({ 
      success: true, 
      data: {
        partnerBonus,
        totalBonusRate,
        enhancedRate
      }
    });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});
```
