## 1. 시스템 개요 (Executive Summary)

본 명세서는 BitWish Network의 데이터베이스에 산재된 파편화된 정보를 단일 시야(Single View)로 통합하고, 시스템 전체의 무결성을 전수 검증하는 **[Unified Query & Audit Engine]**의 기술적 규격을 정의합니다. 본 시스템은 단순 조회를 넘어, 복잡한 추천 계층 구조를 시각화하고 실시간 채굴 보정치를 역산하여 **'수학적으로 증명된 진실'**만을 추출하는 인프라스트럭처의 중추 역할을 수행합니다.

---

## 2. 핵심 기술 1: 3-Way 컬렉션 어그리게이터 (Atomic Join)

파편화된 유저의 정보를 원자적(Atomic)으로 통합하는 기술입니다.

*   **동시성 병렬 로딩:** `User`(가입), `MiningState`(채굴), `BonusRecord`(보너스)라는 3개의 독립된 컬렉션을 `Promise.all` 기반의 비동기 병렬 파이프라인으로 동시에 긁어모읍니다.
*   **관용적 검색 엔진 (Tolerance Engine):** 지갑 주소의 대소문자 오입력, 앞뒤 공백 등을 `RegExp(Ignore Case)` 필터로 자동 정규화하여, 사용자(재단)의 사소한 실수에도 100% 정확한 데이터를 찾아냅니다.
*   **쉬운 설명:** "여러 곳에 흩어져 있는 유저의 신상 정보, 일한 기록, 보너스 장부를 단 0.1초 만에 한 페이지로 합쳐서 보여주는 강력한 돋보기입니다."

---

## 3. 핵심 기술 2: 실시간 델타 보정 연산기 (Live Delta Calculator)

DB의 정적 데이터와 현재 시점의 동적 데이터를 결합하는 고차원 연산 기술입니다.

*   **시간 차(Delta) 추출:** 서버의 현재 시각과 유저의 마지막 싱크 시간(`lastSyncTime`)의 차이를 밀리초(ms) 단위로 정밀 추출합니다.
*   **실시간 투영:** 추출된 시간 차이에 유저 고유의 채굴률을 곱하여, DB에 아직 저장되지 않은 **'미기록 채굴량'**을 `Decimal.js` 50자리 정밀도로 즉시 계산하여 합산 출력합니다.
*   **쉬운 설명:** "DB에 기록된 과거의 숫자만 보는 것이 아니라, 유저가 지금 이 순간에도 채굴하고 있는 실시간 수량까지 계산기에 넣어서 보여주는 기술입니다."

---

## 4. 핵심 기술 3: 계층 구조 시각화 트리 (Referral Tree Tracer)

복잡하게 얽힌 추천인 관계를 인간이 이해하기 쉬운 구조로 변환하는 기술입니다.

*   **재귀적 트리 렌더링:** `ㄴ`, `├─`, `└─` 등의 ASCII 심볼 엔진을 활용하여 부모와 자식의 관계를 계단식 그래프로 시각화합니다.
*   **무결성 필터 (Ghost Detection):** 각 노드를 순회할 때 추천 코드가 장부와 일치하는지 실시간 대조하여, 가입은 되었으나 보상 체계에서 누락된 **'유령 자식'**을 `✅/❌/⚠️` 아이콘으로 즉각 식별합니다.
*   **쉬운 설명:** "가계도처럼 누가 누구를 추천했는지 한눈에 보여주고, 혹시라도 연결이 끊긴 유저가 있다면 빨간 경고등을 켜주는 똑똑한 추적기입니다."

---

## 5. 핵심 기술 4: 시스템 전수 감사 로직 (Full System Audit)

1명부터 수천만 명까지, 전 가입자의 무결성을 자동으로 검증하는 감사 기술입니다.

*   **정책 기반 기대값 산출:** 공식 정책(`2% + n*2%`)을 알고리즘화하여, 각 유저가 마땅히 받아야 할 **[이론적 기대치]**를 산출합니다.
*   **실시간 동기화 감사(Real-time Parity Audit):** 마이닝 모달의 실시간 닻(Anchor) 연산 결과가 랭킹 및 어드민의 `LiveBoost` 데이터와 단 10^-50 (소수점 50자리)의 오차도 없이 일치하는지 전수 검증합니다.
*   **Zero-Leak 검증:** [이론적 기대치]와 [실제 DB 수치]를 1:1로 매칭 분석하여, 단 10^-50의 오차라도 발생할 경우 즉시 '결함 등급'으로 분류하여 보고합니다.
*   **CLI 인터페이스:** `npm run admin`, `npm run audit` 명령어로 비전문가도 즉시 시스템 전체를 스캔할 수 있는 운영 편의성을 제공합니다.
*   **쉬운 설명:** "재단이 정한 규칙대로 모든 유저의 돈이 정확하게 계산되고 있는지, 특히 마이닝 화면과 게시판의 숫자가 한 치의 오차도 없이 일치하는지 전 세계 모든 유저를 대상으로 단 몇 분 만에 전수 검사하는 디지털 감사관입니다."
