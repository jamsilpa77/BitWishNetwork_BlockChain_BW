## 3. 완벽한 수정 계획 (단계별 작업 순위)

### **우선순위 1 (최고 긴급): `MiningStatusModal.tsx` 서버 절대 복종 개조**

#### 작업 1-1: 서버 데이터 직접 반영 로직 추가
**파일:** `src/components/MiningStatusModal/MiningStatusModal.tsx`  
**목표:** 서버 응답의 `rewardStorage`, `bonusStorage`, `totalBonusRate`를 **직접** 화면 상태에 설정

**수정 내용:**
```tsx
const initializeData = async () => {
  const userData = await apiService.getUserStatus(walletAddress);
  
  if (!userData || !userData.data) return;
  
  // [FIX] 서버 데이터 최우선 반영
  const serverRewardStorage = parseFloat(userData.data.rewardStorage || 0);
  const serverBonusStorage = parseFloat(userData.data.bonusStorage || 0);
  const serverTotalBonusRate = userData.miningState?.totalBonusRate || 0;
  const serverReferralCount = userData.miningState?.referralCount || 0;
  
  // [FIX] 로컬 스토리지 동기화 (서버 → 로컬, 단방향)
  if (userData.data.referees && Array.isArray(userData.data.referees)) {
    const localData = JSON.parse(localStorage.getItem('bw_referral_data') || '{}');
    localData[walletAddress] = {
      referralCode: userData.data.referralCode,
      referredUsers: userData.data.referees,
      rewardStorage: serverRewardStorage,
      bonusStorage: serverBonusStorage
    };
    localStorage.setItem('bw_referral_data', JSON.stringify(localData));
  }
  
  // [FIX] 화면 상태에 서버 데이터 직접 설정
  setMiningState(prev => ({
    ...prev,
    referralCount: serverReferralCount,
    referralBonusRate: serverTotalBonusRate,
    rewardStorage: serverRewardStorage,
    bonusStorage: serverBonusStorage
  }));
};
```
