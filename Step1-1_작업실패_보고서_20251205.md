## 🔍 왜 이런 오류가 발생했는가?

### **근본 원인 1: TargetContent 불일치**

**제가 지정한 TargetContent:**
```tsx
                                            <h4>{selectedMonth}월 가입자 목록</h4>
```

**실제 파일 내용:**
```tsx
                                             <h4>{selectedMonth}월 가입자 목록</h4>
```

**차이점:**
- 공백 문자 수가 다름
- 제가 지정: 44개 공백
- 실제 파일: 45개 공백 (또는 탭 문자 혼용)

**결과:**
- replace_file_content 도구가 정확한 위치를 찾지 못함
- "We did our best to apply changes despite some inaccuracies" 메시지
- 도구가 임의로 비슷한 위치에 코드 삽입 시도
- 파일 구조 완전 손상

---

### **근본 원인 2: replace_file_content 도구의 한계**

**도구의 동작 방식:**
1. TargetContent를 파일에서 검색
2. 정확히 일치하는 문자열 찾기
3. 찾지 못하면 "best effort" 시도
4. 결과: 예측 불가능한 수정

**문제점:**
- 공백 문자 하나라도 다르면 실패
- `\r\n` vs `\n` 차이도 실패 원인
- 탭 vs 스페이스 차이도 실패 원인
- 실패 시 "최선을 다해" 수정 시도 → 파일 손상
