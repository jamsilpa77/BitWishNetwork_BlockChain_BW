# 최신형 커뮤니티 게시판 시스템 설계

요청한 구조를 기준으로 요즘 많이 사용하는 커뮤니티 스타일을 벤치마킹해서 아래 기능들을 포함한 “실전형 커뮤니티 게시판 시스템” 구조로 설계합니다.

벤치마킹 스타일:

* DC Inside 스타일의 빠른 게시판
* Reddit 스타일 추천/이모지 반응
* Blind 스타일 인기글 시스템
* FM Korea 스타일 카테고리형 게시판

```prisma
model User {
  id        Int      @id @default(autoincrement())
  email     String   @unique
  nickname  String
  password  String
  role      String   @default("USER")
  createdAt DateTime @default(now())

  posts     Post[]
  comments  Comment[]
}
```

---

## Post 테이블

```prisma
model Post {
  id          Int       @id @default(autoincrement())
  title       String
  content     String
  category    String
  thumbnail   String?
  views       Int       @default(0)

  likeCount   Int       @default(0)
  dislikeCount Int      @default(0)
  heartCount  Int       @default(0)
  funnyCount  Int       @default(0)

  hotScore    Float     @default(0)

  isNotice    Boolean   @default(false)

  createdAt   DateTime  @default(now())

  authorId    Int
  author      User      @relation(fields: [authorId], references: [id])

  comments    Comment[]
}
```

---

## Comment 테이블

```prisma
model Comment {
  id        Int      @id @default(autoincrement())
  content   String

  postId    Int
  userId    Int

  createdAt DateTime @default(now())

  post      Post @relation(fields: [postId], references: [id])
  user      User @relation(fields: [userId], references: [id])
}
```

# 리액션 시스템 로직

## Reaction 타입

```ts
type ReactionType =
  | "LIKE"
  | "DISLIKE"
  | "HEART"
  | "FUNNY";
```

---

## 리액션 처리

```ts
async function reactPost(postId, type) {
  switch(type) {
    case "LIKE":
      post.likeCount++;
      break;

    case "DISLIKE":
      post.dislikeCount++;
      break;

    case "HEART":
      post.heartCount++;
      break;

    case "FUNNY":
      post.funnyCount++;
      break;
  }

  post.hotScore =
      post.likeCount * 3 +
      post.heartCount * 5 +
      post.funnyCount * 2 -
      post.dislikeCount * 2;
}
```

# Pagination 컴포넌트

```tsx
export default function Pagination({
  currentPage,
  totalPages,
  onPageChange
}) {
  return (
    <div className="flex gap-2">
      {currentPage > 1 && (
        <button onClick={() => onPageChange(currentPage - 1)}>
          이전
        </button>
      )}

      {[...Array(totalPages)].map((_, i) => (
        <button
          key={i}
          onClick={() => onPageChange(i + 1)}
        >
          {i + 1}
        </button>
      ))}

      {currentPage < totalPages && (
        <button onClick={() => onPageChange(currentPage + 1)}>
          다음
        </button>
      )}
    </div>
  );
}
```
