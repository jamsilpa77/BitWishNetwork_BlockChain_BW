# 02- BitWish Network 기술 명세서 (Technical Specification)

**버전:** v1.0.0 (Phase 4 융합 완료)
**작성일자:** 2026년 3월 21일
**대상:** 시스템 아키텍트, 블록체인 개발자, 인프라 엔지니어

## 1. 시스템 아키텍처 (System Architecture)

### 1.1 하이브리드 이중 장약 방식 (Hybrid Dual-Commit)
BitWish Network는 고속 웹 트랜잭션과 무결점 블록체인 검증을 동시에 달성하기 위해 **[JSON DB] + [Blockchain Core]**의 이중 장약 방식을 채택하고 있습니다.
- **Master Data:** `referrals.json` (실시간 포인트 및 지갑 잔액 원본)
- **Security Layer:** `BitWishBlockchain.ts` (블록체인 검증 및 마이닝 장부)
- **Consistency Algorithm:** `verifyAndMineTransaction` 브릿지 함수를 통해 엔진의 '성공' 응답 시에만 JSON 숫자가 갱신되는 원자성(Atomicity)을 보장합니다.

### 1.2 50자리 초정밀 부동소수점 연산 (Decimal Precision)
- **라이브러리:** `Decimal.js` (50자리 정밀도 설정)
- **목적:** 비트코인(8자리) 이상의 초정밀 금융 연산을 수행하며, 전송 및 채굴 보상 계산 시 부동소수점 오차를 0%로 수렴시킵니다.
- **적용:** `BitWishTransaction`, `BitWishAccount`, `BitWishBlock` 전 영역 적용.

## 2. 블록체인 코어 기술 사양

### 2.1 합의 알고리즘 (Consensus)
- **방식:** BitWish PoW (Proof of Work)
- **해시 알고리즘:** BitWish-256 (SHA-256 기반 암호화 프로토콜)
- **난이도 조절:** 2,016 블록마다 동적 난이도 조절(Target Adjustment)을 수행하며, 현재 제네시스 설정은 10초당 1블록 생성을 목표로 합니다.

### 2.2 블록 및 트랜잭션 구조
- **BitWishBlock:**
    - `Header`: 버전, 이전 블록 해시, 머클 루트, 타임스탬프, 난이도, 논스, 블록 높이, 검증자 주소, 보상액 정보 포함.
    - `Data`: `BitWishTransaction` 리스트 (최대 1,000개/블록)
- **BitWishTransaction:**
    - `Hash`: 트랜잭션 고유 식별자 (BitWish-256)
    - `Signature`: 디지털 서명 검증을 통한 보안 전송 보장
    - `Gas System`: `gasLimit` 및 `gasPrice` 기반의 거래 수수료 및 무한 루프 방지 시스템 완비.

### 2.3 BWVM (BitWish Virtual Machine)
- **개요:** 스마트 컨트랙트 실행을 위한 전용 가상 머신.
- **스택 기반 실행:** 바이트코드를 해석하고 실행하며 계정의 `storageRoot` 상태를 변경하여 튜링 완전성(Turing Completeness)을 지원할 준비가 되어 있습니다.

## 3. 백엔드 통합 및 인프라 사양

### 3.1 모듈 인스턴스화 (Singleton)
- **위치:** `BitWishNetwork_MiningSystem/server/index.ts`
- **로직:** 백엔드 프로세스 가동 시 `bwChainCore`가 유일한 객체로 생성되어, 모든 API 라우터(`stats.ts` 등)에서 공유됩니다.
- **포트 정보:**
    - **API 전송:** 5001 (Express JS)
    - **프론트 웹:** 5000 (React)
    - **P2P 노드:** 4002 (독립 기동용)

### 3.2 데이터베이스 하이브리드 저장 (Storage)
- **MongoDB:** `bitwish_mining` (추천/채굴 통계), `bitwish_network` (블록/계정 상태 데이터)
- **LevelDB/State Trie:** 지갑 상태의 위변조 방지를 위한 머클 패트리샤 트리(Merkle Patricia Trie) 로직 내장.
