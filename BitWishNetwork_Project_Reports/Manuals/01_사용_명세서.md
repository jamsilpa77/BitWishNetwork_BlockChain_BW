# 01- BitWish Network 사용 명세서 (User & Operator Guide)

**버전:** v1.0.0 (Phase 4 융합 완료판)
**작성일자:** 2026년 3월 21일
**대상:** BitWish Network 사용자 및 시스템 운영자

---

## 1. BitWish Network 개요
BitWish Network는 기존의 웹 기반 채굴 시스템과 차세대 독립 블록체인 엔진이 결합된 **하이브리드 메인넷 생태계**입니다. 사용자는 안전한 지갑 환경에서 BW 토큰을 채굴하고, 3~4초 내의 정밀한 블록체인 검증을 거쳐 P2P 전송을 수행할 수 있습니다.

---

## 2. 지갑(Wallet) 인터페이스 가이드

### 2.1 지갑 생성 및 보관
- **지갑 주소 발급:** `BW`로 시작하는 42자리 독립 주소가 발급됩니다.
- **보안 인증:** 니모닉(Mnemonic) 및 개인키(Private Key)는 로컬 환경에 암호화되어 저장되며, 분실 시 복구가 불가능하므로 철저히 관리해야 합니다.
- **인증 세션:** 보안을 위해 10분 단위의 활동 세션이 적용되어 자동 로그아웃을 지원합니다.

### 2.2 P2P 코인 전송 (하이브리드 엔진 작동)
- **전송 프로세스:** 송금 요청 시 블록체인 엔진이 즉시 개입하여 발신자의 잔액 무결성을 검증합니다.
- **마이닝 딜레이:** 모든 전송은 약 3~4초간의 '마이닝(장부 기록)' 과정을 거칩니다. 이 과정에서 해시가 생성되며 블록체인 장부에 영구 박제됩니다.
- **성공 확인:** 엔진의 검증이 완료된 후에만 실제 숫자가 갱신되어 전송이 완료됩니다.

---

## 3. 실시간 채굴 및 스테이킹 시스템

### 3.1 마이닝 보너스 (Mining & Bonus)
- **출석 보너스:** 매일 9시(KST) 기준 출석 체크 시 5%의 고정 채굴 가중치가 부여됩니다.
- **추천 보너스:** 지인 추천 시 추천인과 가입자 모두에게 정밀한 BW 보너스가 즉시 배분됩니다.
- **실시간 데이터 동기화:** 홈페이지와 마이닝 모달, 지갑 페이지의 수치는 1초 단위로 오차 없이 실시간 동기화됩니다.

### 3.2 대시보드(Dashboard) 활용
- **전액 모니터링:** 총 공급량(210억), 실시간 발행량, 잔여 발행량을 초정밀 수치로 확인합니다.
- **발행률 상세 노출:** 발행률이 0.00%로 보일 경우, 해당 숫자에 마우스를 올리면(Hover) 소수점 8자리까지의 정밀한 블록체인 진행률이 표시됩니다.
- **네트워크 상태:** `CONNECTED` 표시를 통해 현재 블록체인 엔진과의 연결 상태를 실시간으로 체크할 수 있습니다.

---

## 4. 보안 가이드라인

- **2차 비밀번호:** P2P 전송 및 지갑 접근 시 2차 보안 비밀번호 입력이 필수적입니다.
- **다크 모드 지원:** 장시간 채굴 모니터링 시 사용자의 시력 보호를 위해 고품격 다크 모드를 상시 지원합니다.
- **위변조 감지:** 시스템은 초당 수억 번의 해시 연산을 통해 장부의 위변조를 감지하며, 이상 징후 발생 시 전송을 자동으로 차단합니다.
