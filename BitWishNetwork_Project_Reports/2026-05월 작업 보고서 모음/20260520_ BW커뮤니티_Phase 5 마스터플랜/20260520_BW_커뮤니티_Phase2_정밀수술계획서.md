### 🟢 [수술 부위 2] `src/components/BWCommunityWindow/BWCommunityWindow.tsx` (프리미엄 UI 전면 개편)
*   **작업 목적:** QA 깡통 버튼 삭제, 글래스모피즘(Glassmorphism) 기반 디자인, 구글 연동 폼 마련, 6대 카테고리, 마이크로 애니메이션(Pop) 적용.
*   **기존 코드 로직의 문제점:** 
    - 평면적인 단색 CSS 변수 (다크모드 지원이 미흡)
    - 어설픈 `handleQALogin` (유저/어드민 자동생성) 함수와 버튼 존재
    - GENERAL, TECH 2개의 하드코딩된 카테고리 탭

*   **수복될 신규 코드 (핵심 요약):**

    **1. 글래스모피즘 및 애니메이션 CSS 렌더링 주입:**
    ```css
    /* 기존 밋밋한 단색 배경에서 반투명 블러 효과(Glassmorphism)로 교체 */
    --bw-bg-primary: rgba(17, 24, 39, 0.7);
    --bw-glass-border: rgba(255, 255, 255, 0.1);
    backdrop-filter: blur(12px);

    /* 마이크로 인터랙션 (버튼 타격감) */
    @keyframes popBouncing {
        0% { transform: scale(1); }
        50% { transform: scale(1.15); }
        100% { transform: scale(1); }
    }
    .bw-reaction-btn { transition: all 0.2s; }
    .bw-reaction-btn:active { animation: popBouncing 0.3s ease; }
    ```

    **2. 쓰레기 기능(QA 버튼) 삭제 및 구글 로그인 UI 신설:**

### 🛠️ 수술 완료 부위 및 검증 내역

1. **`communityClient.ts` 통신로 격리 복구 완료**
   - 기존 마이닝의 통신을 우회하여, `http://localhost:5001/api/bw-community` 전용 플러그인 라우터로 경로를 100% 분리 수복했습니다.
2. **`BWCommunityWindow.tsx` 초프리미엄 렌더링 수술 완료**
   - **QA 깡통 버튼 영구 삭제:** 허술했던 강제 로그인/자동생성 버튼을 코어에서 완전히 도려냈습니다.
   - **Glassmorphism 및 다크모드 완벽 이식:** 반투명 유리 질감(`backdrop-filter`)과 3D 그림자를 적용한 다크모드/라이트모드 에셋을 성공적으로 주입했습니다.
   - **6대 카테고리 & 마이크로 애니메이션:** `[유머, 정보, 자유, 질문, 게임, 익명]` 알약 형태 탭과 `Skeleton Loading`, 리액션 애니메이션(`popBouncing`)을 장착했습니다.
   - **구글 로그인 안전 대기(Mocking) 처리:** 터미널 미사용 원칙에 입각하여 프리미엄 구글 로그인 버튼 레이아웃만 렌더링해 두어 규칙 위반을 원천 차단했습니다.
3. **`locales/community_i18n.ts` 4개국어 언어팩 업데이트 완료**
   - 6대 카테고리에 대한 4개국어(한, 영, 일, 중) 지원 로직을 완벽하게 주입 완료했습니다.
