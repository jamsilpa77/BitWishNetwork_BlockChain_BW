## 🖥️ 1. 6공정: 전체 React 프론트엔드 독립 뷰(View) 설계

**[작업 위치 (신규 생성)]**
*   `c:\BitWishNetwork_BlockChainMainnet\BitWishNetwork_MiningSystem\src\components\BWCommunityWindow\` (해당 폴더 내부에서만 모든 작업 진행)

**[기존 코드 로직 (현재 상태)]**
*   기존 마이닝 시스템 프론트엔드는 채굴 데이터와 타이머 렌더링에 집중되어 있으며, 커뮤니티를 위한 컴포넌트가 없음.

**[수술/신설 로직 요약]**
*   **`BWCommunityWindow.tsx` (핵심 껍데기):** 전역 모달(Global Modal)을 절대 사용하지 않습니다! 마이닝 페이지 한 쪽에 드래그 앤 드롭(Drag & Drop)으로 띄워놓을 수 있는 '독립된 로컬 팝업 컴포넌트'를 신설합니다.
*   **`api/communityClient.ts`:** 마이닝 API와 섞이지 않도록, Phase 2에서 만든 커뮤니티 전용 백엔드(포트)와 통신하는 독립적인 API 호출 유틸리티를 만듭니다.
*   **로컬 상태 격리:** 게시글 목록(`BoardList`), 상세 보기(`PostDetail`), 댓글(`CommentArea`) 등의 상태(State)는 Redux나 Zustand 같은 전역 상태 관리 라이브러리에 **절대** 올리지 않고, 오직 `BWCommunityWindow` 내부의 `useState`와 로컬 `Context`로만 가둬둡니다.

## 🎨 2. 7공정: Tailwind 기반 최신 UI, 자체 다크모드, 4개국어 다국어 이식

**[작업 위치 (신규 생성)]**
*   `BWCommunityWindow\hooks\useCommunityTheme.ts` (자체 다크모드 훅)
*   `BWCommunityWindow\locales\community_i18n.ts` (다국어 엔진)
*   `BWCommunityWindow\components\UI\...` (카드형 디자인 컴포넌트들)

**[수술/신설 로직 요약]**
*   **자체 다크/나이트 모드 (독립성 보장):** 마이닝 홈페이지가 라이트 모드이더라도, 유저가 커뮤니티 창의 '달(🌙) 버튼'을 누르면 **커뮤니티 창 내부만 즉시 다크모드**로 변환되도록 철저히 스코프(Scope)를 제한한 CSS 테마를 적용합니다.
*   **4개국 다국어 번역:** 한국어, 영어, 일본어, 중국어 4개국어 텍스트 JSON 데이터를 분리하여, 커뮤니티 내부의 언어 선택기(Select)를 통해 텍스트가 즉각 반응(번역)하도록 설계합니다.
*   **Tailwind UI 디자인:** 레딧+블라인드 스타일의 부드러운 Hover 애니메이션과 이모지 버튼을 구현하며, CSS 충돌 방지를 위해 커뮤니티 내부 클래스에만 적용되도록 캡슐화합니다.

## ⚠️ 3. 절대 준수 규칙 크로스체크
1.  **전역 상태 / 전역 모달 / 공통 함수 사용 배제 (100%):** 커뮤니티 윈도우는 우주를 떠도는 독립된 우주선처럼 설계됩니다. 부모 컴포넌트(마이닝)와 그 어떤 상태도 주고받지 않습니다.
2.  **기존 마이닝 로직/파일 침범 제로 (100%):** `BWCommunityWindow`라는 신규 폴더 내부에서만 작업하므로, 기존 마이닝 UI 소스코드에 삭제나 변경이 발생할 확률은 0%입니다.
3.  **다국어 및 다크모드 필수 적용 (100%):** 요구하신 4개국어 시스템과 다크모드를 윈도우창 전용으로 완벽히 기획했습니다.
