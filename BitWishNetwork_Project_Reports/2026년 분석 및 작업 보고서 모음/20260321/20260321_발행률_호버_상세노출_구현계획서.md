# 20260321_발행률_호버_상세노출_구현계획서

**작성일자:** 2026년 3월 21일
**작성자:** Antigravity 
**수정 대상:** 실시간 BW 발행률(Issuance Rate) UI 컴포넌트

---

## 1. 개요 및 목적
현재 발행량(20 BW)이 총 공급량(210억 BW)에 비해 현저히 낮아 대시보드상 '0.00%'로 표시되어 정지된 것으로 보일 수 있는 UI를 개선합니다. 평시에는 디자인적 깔끔함을 유지(0.00%)하되, 마우스 호버 시 실제 블록체인 연산 수치(소수점 8자리)를 상세히 보여줌으로써 시스템의 무결함을 증명하는 것을 목적으로 합니다.

---

## 2. 상세 수정 작업 계획 (위치 및 로직)

### 🛠️ [작업 1] 백엔드 데이터 전송 규격 정밀화 
- **파일:** `BitWishNetwork_MiningSystem/server/routes/stats.ts`
- **로직:** 수치를 강제로 자르지 않고, 프론트엔드에서 연산 가능한 원본 문자열(Decimal string) 형태로 전송합니다. 
```typescript
// [수정 전]
issuanceRate: issuanceRate.toFixed(2)

// [수정 후] 
issuanceRate: issuanceRate.toString() 
```

### 🛠️ [작업 2] 홈페이지 UI 호버 정밀 데이터 주입
- **파일:** `BitWishNetwork_MiningSystem/src/components/HomePage/HomePage.tsx`
- **로직:** 발행률 카드 수치 태그에 `title` 속성(Native Tooltip)을 부여하여 정밀 정보를 노출합니다.
```tsx
// [수정 전]
<div className="card-value">{globalStats.issuanceRate}%</div>

// [수정 후]
<div 
    className="card-value" 
    title={`${new Decimal(globalStats.issuanceRate || 0).toFixed(8)}%`}
    style={{ cursor: 'pointer' }}
>
    {new Decimal(globalStats.issuanceRate || 0).toFixed(2)}%
</div>
```

---

## 3. 작업 순서 및 무결성 보장

1. 백엔드 통신 데이터 전송 정밀도를 소수점 8자리 이상으로 확장 적용합니다.
2. 홈페이지 리액트 컴포넌트의 발행률 노출 수치에 8자리 호버 로직을 정확히 이식합니다.
3. 다른 컴포넌트나 지갑 정보는 단 0.1%도 수정하지 않으며 오직 해당 수치 표시 기능만 개선합니다.
