## 1. 개요 (Overview)
현재 [나의 지갑]에서 [마이닝 시작]을 클릭할 때, 지갑 주소 데이터가 전달되지 않아 마이닝 페이지가 작동하지 않는 현상이 발생합니다. 이를 해결하기 위해 데이터를 전달하고 받는 파이프라인을 복구합니다.

---

## 2. 작업 상세 (Step-by-Step)

### [단계 1] `MyWalletModal.tsx` 수정 (데이터 발신)

**1. 인터페이스 수정**
*   **위치:** `MyWalletModalProps` 인터페이스 정의부
*   **변경:**
    ```typescript
    onOpenMining?: () => void;
    // ↓ 변경 ↓
    onOpenMining?: (address: string) => void;
    ```

**2. 핸들러 호출 수정**
*   **위치:** `[⛏️ 마이닝 시작]` 버튼 `onClick` 이벤트 내부
*   **변경:**
    ```typescript
    onOpenMining();
    // ↓ 변경 ↓
    onOpenMining(walletAddress); // 현재 컴포넌트가 가진 주소를 인자로 전달
    ```

**3. 중복 UI 제거 (Cleanup)**
*   **위치:** 로그아웃 버튼 하단 (`header-right-group` 내부)
*   **작업:** 실수로 남겨둔 `2차 비번 초기화` 버튼 코드 블록 삭제. (이미 파란 헤더로 옮겨졌으므로 중복임)

---

### [단계 2] `HomePage.tsx` 수정 (데이터 수신 및 등록)

**1. 핸들러 구현 수정**
*   **위치:** `<MyWalletModal ... />` 컴포넌트 선언부
*   **변경:**
    ```typescript
    onOpenMining={() => {
        setIsMyWalletModalOpen(false);
        setIsMiningModalOpen(true);
    }}
    // ↓ 변경 ↓
    onOpenMining={(address) => {
        // [핵심] 받은 주소를 시스템 전체 인증 주소로 등록
        setAuthenticatedAddress(address);
        setIsMyWalletModalOpen(false);
        setIsMiningModalOpen(true);
    }}
    ```
