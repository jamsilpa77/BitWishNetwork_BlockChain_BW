### 요구사항 B: 즉시 영구 고정 적용 (Step 3041)
- 추천 가입 발생 시 → 보상(1BW) + 보너스(2%) **즉시 지급**
- 기본 시간/일일 보상률에 **즉시 반영**
- **채굴 중이라도 실시간 반영** (WebSocket 또는 Polling)

### 요구사항 C: 피라미드 무한 확장 (Step 3041, 3061)
- A→B→C→D 다단계 연결 가능
- 각 단계마다 보상 자동 계산 및 상위 노드에 전파
- 서버 로직 검증 (현재 미검증)

### 요구사항 D: 보관함 완전 분리 (Step 3044)
- **추천 보상 보관함** (`rewardStorage`): 1BW 보상 누적
- **추천 보너스 보관함** (`bonusStorage`): 2% 채굴 보너스 누적
- 두 값이 독립적으로 표시 및 저장

### 요구사항 E: 가입자 보너스 적용 (Step 3044)
- `isInvited === true`이면 무조건 +2% 보너스
- 시간당/일일 보상률에 즉시 반영

#### 작업 2-1: `server.js` - `bonusStorage` 계산 로직 추가
**파일:** `server.js`

**목표:** `rewardStorage`(1BW 보상)와 `bonusStorage`(2% 채굴 누적) 분리

**수정 내용:**
```javascript
// /api/user/status/:userId 엔드포인트 수정
this.app.get('/api/user/status/:userId', (req, res) => {
  const userId = req.params.userId;
  const normalizedUserId = userId.toLowerCase().trim();
  
  let userNode = this.referrals.find(r => r.owner.toLowerCase().trim() === normalizedUserId);
  
  if (!userNode) {
    return res.json({
      success: true,
      data: {
        userId: userId,
        status: 'new',
        referralCode: null,
        referees: [],
        rewardStorage: '0.0000',
        bonusStorage: '0.0000'
      },
      miningState: {
        isMining: false,
        referralCount: 0,
        referralBonusRate: 0,
        accumulatedReward: '0.00000000',
        isInvited: false,
        totalBonusRate: 0
      }
    });
  }

  const myRefereeCount = userNode.referees.length;
  const isInvited = userNode.referrerId ? true : false;
  
  const myBonusRate = myRefereeCount * 2.0;
  const invitedBonusRate = isInvited ? 2.0 : 0;
  const totalBonusRate = myBonusRate + invitedBonusRate;

  res.json({
    success: true,
    data: {
      userId: userId,
      status: 'active',
      referralCode: userNode.code,
      referees: userNode.referees,
      
      // [FIX] 보관함 분리
      rewardStorage: (userNode.rewardStorage || 0).toFixed(4),
      bonusStorage: (userNode.bonusStorage || 0).toFixed(4)
    },
    miningState: {
      isMining: false,
      referralCount: myRefereeCount,
      referralBonusRate: totalBonusRate,
      accumulatedReward: '0.00000000',
      
      isInvited: isInvited,
      totalBonusRate: totalBonusRate
    }
  });
});
```

```javascript
// [New] 추천 상태 변경 알림 API
this.app.get('/api/referral/check-updates/:userId', (req, res) => {
  const userId = req.params.userId;
  const normalizedUserId = userId.toLowerCase().trim();
  
  const userNode = this.referrals.find(r => r.owner.toLowerCase().trim() === normalizedUserId);
  
  if (!userNode) {
    return res.json({ hasUpdate: false });

  // 마지막 조회 이후 변경 확인 (간단한 구현: referees 개수 비교)

    currentReferralCount: currentCount,

  });
```

```tsx
useEffect(() => {
  if (!isMining) return;
  
  const checkUpdates = async () => {
    const updates = await apiService.checkReferralUpdates(walletAddress);
    if (updates.hasUpdate && updates.currentReferralCount !== miningState.referralCount) {
      // 서버 재조회
      const userData = await apiService.getUserStatus(walletAddress);
      setMiningState({
        referralCount: userData.miningState.referralCount,
        referralBonusRate: userData.miningState.totalBonusRate,
        // ... 나머지 필드
      });
      
      // 보상률 재계산
      const newHourlyRate = 0.25 * (1 + userData.miningState.totalBonusRate / 100);
      setHourlyRate(newHourlyRate);
    }
  };
  
  const timer = setInterval(checkUpdates, 30000); // 30초마다
  return () => clearInterval(timer);
}, [isMining, walletAddress]);
```
