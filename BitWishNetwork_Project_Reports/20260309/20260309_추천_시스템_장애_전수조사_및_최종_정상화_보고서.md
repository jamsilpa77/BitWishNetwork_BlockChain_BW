- **수정 코드:**
```typescript
public generateReferralCode(address: string): string {
    // 관리자 하드코딩 (9DC5 보존)
    if (address === 'BW9F5FF090231236037F250A523B4FC320FB44BFA8') return 'REF9F5FF0909DC5';
    const prefix = address.substring(2, 10).toUpperCase();
    const suffix = address.substring(address.length - 4).toUpperCase();
    return `REF${prefix}${suffix}`; // 15자리 정상화
}
```

### **(2) 백엔드: 실적 업데이트 로직 강화**
- **파일:** `server/controllers/UserController.ts` (L188-225)
