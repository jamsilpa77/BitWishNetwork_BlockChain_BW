## 2. 최종 적용 코드 상세

### 2-1. `MiningAuthModal.tsx`: 실시간 주소 감지 시스템
**기능:** 모달이 열려있는 동안에도 지갑이 바뀌면 즉시 주소를 갱신함.

```typescript
// c:\BitWishNetwork_BlockChainMainnet\BitWishNetwork_MiningSystem\src\components\MiningAuthModal\MiningAuthModal.tsx

// [수정] useEffect 내부 로직 강화
useEffect(() => {
    // 1. 초기 로드 시 주소 가져오기
    const loadWalletAddress = () => {
        const savedWalletData = localStorage.getItem('bw_wallet_data');
        if (savedWalletData) {
            try {
                const { address: savedAddress } = JSON.parse(savedWalletData);
                setAddress(savedAddress || '');
            } catch {
                setAddress('');
            }
        } else {
            setAddress('');
        }
    };

    if (isOpen) {
        // 기존 유예 타임 체크 (생략...)
        
        loadWalletAddress(); // 즉시 실행
        setPassword('');
        setError('');
    }

    // 2. [보완] 실시간 변경 감지 (지갑 바뀌면 모달도 바뀜)
    const handleStorageChange = () => loadWalletAddress();
    window.addEventListener('storage', handleStorageChange);
    window.addEventListener('BW_WALLET_UPDATE', handleStorageChange); // 커스텀 이벤트 지원

    // 3. 클린업 (메모리 누수 방지)
    return () => {
        window.removeEventListener('storage', handleStorageChange);
        window.removeEventListener('BW_WALLET_UPDATE', handleStorageChange);
    };
}, [isOpen]);
```

### 2-2. `WalletService.ts`: 방어적 자동 등록 시스템
**기능:** 데이터가 완벽할 때만 서버에 등록(Upsert)하여 무결성 보장.

```typescript
// c:\BitWishNetwork_BlockChainMainnet\BitWishNetwork_MiningSystem\src\services\BlockchainService\WalletService.ts

public async verifySecondPassword(address: string, password: string): Promise<boolean> {
    const localWallet = this.getWalletFromStorage();
    let storedPasswordHash = '';
    
    // ... (기존 해시 확보 로직 동일) ...

    if (!storedPasswordHash) return false;

    // 비밀번호 검증
    const [salt, storedHash] = storedPasswordHash.split(':');
    const currentHash = this.generateHash(password, salt);
    if (currentHash !== storedHash) return false;

    // =================================================================
    // [보완] 4. 검증 성공 후: "방어적" 데이터 동기화
    // =================================================================
    try {
        const serverCheck = await apiService.getUserStatus(address);
        
        // 서버에 데이터가 없을 때만 실행
        if (!serverCheck || !serverCheck.success || !serverCheck.data) {
            console.log('[WalletService] Auto-Register triggered.');

            // [Critical Check] 중요 데이터 누락 여부 확인
            if (localWallet && localWallet.address === address 
                && localWallet.encryptedMnemonic // 시드 구문 필수
                && localWallet.publicKey) {      // 공개키 필수

                await apiService.registerUser({
                    walletAddress: address,
                    publicKey: localWallet.publicKey,
                    encryptedMnemonic: localWallet.encryptedMnemonic,
                    secondPasswordHash: localWallet.secondPasswordHash, 
                    myReferralCode: localWallet.myReferralCode,
                    referrerCode: localWallet.usedReferralCode || null,
                    ipAddress: '127.0.0.1'
                });

                await apiService.syncMiningData(address, "0");
                console.log('[WalletService] Restoration Complete.');
            } else {
                console.error('[WalletService] Cannot auto-register: Missing critical local data.');
                // 데이터가 깨진 경우 등록하지 않음 (쓰레기 데이터 생성 방지)
            }
        }
    } catch (e) {
        console.warn('[WalletService] Sync warning:', e);
    }

    return true;
}
```
