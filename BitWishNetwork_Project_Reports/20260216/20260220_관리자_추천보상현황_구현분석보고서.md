## 3. 구현 계획 (To-Be)

### 3-1. Frontend 구현 (`AdminPage.tsx`)
1.  **탭 추가:** `<button>` "추천 보상 현황" 추가 (`activeTab === 'rewardStatus'`).
2.  **전체 통계 컴포넌트:**
    *   `useEffect`로 탭 진입 시 `GET /api/admin/rewards/total` 호출.
    *   상단에 큰 글씨로 "전체 지급 보상: OOO BW" 표시.
3.  **검색 및 상세 뷰:**
    *   지갑 주소 입력 -> `GET /api/admin/rewards/detail/:address` 호출.
    *   반환된 데이터를 그리드(Grid)와 테이블(Table)로 시각화.

### 3-2. Backend 구현 (`server/routes/admin.ts`)

#### API 1: 전체 지급 현황 조회
*   **URL:** `GET /api/admin/rewards/total`
*   **로직:** `BonusRecord` 전체를 스캔하여 `referralRewardStorage` 값을 모두 더함(`$sum`).
*   **응답:** `{ success: true, totalIssued: number }`

#### API 2: 개별 유저 상세 조회
*   **URL:** `GET /api/admin/rewards/detail/:walletAddress`
*   **로직:**
    *   `User` 테이블 조인 -> `myReferralCode`, `createdAt`(가입일) 조회.
    *   `BonusRecord` 조회 -> `referralRewardStorage`(총 보상), `referralList`(하위 가입자).
*   **응답:**
    ```json
    {
      "walletAddress": "...",
      "myReferralCode": "CODE123",
      "joinDate": "2026-02-20T...",
      "totalReward": "50", // 50 BW
      "referralCount": 50,
      "referralList": [ ... ]
    }
    ```
