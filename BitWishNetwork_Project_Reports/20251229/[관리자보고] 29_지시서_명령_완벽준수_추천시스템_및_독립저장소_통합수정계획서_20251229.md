## 1. 범용 독립 저장소 엔진 (BISE: BitWish Individual Storage Engine) 설계

지시서 라인 22, 23의 "유저는 1명이든 천만명이든 개인 단독 저장소를 구현한다"는 명령을 100% 실체화하는 엔진입니다.

### 가. 저장소 구조 및 자동화 (Line 23 준수)
- **독립성:** 모든 유저는 가입 또는 지갑 생성 시 `database/users/BW[주소].json`이라는 전용 데이터베이스 파일을 발급받음.
- **자동 확장성:** 특정 소수(5인 등)를 위한 코딩이 아닌, 새 유저 유입 시 시스템이 자동으로 표준 규격의 독립 저장소를 즉시 생성하는 **Factory Pattern** 엔진 적용.
- **무결성 사수:** "훼손되어선 안 된다"는 명령에 따라, 파일 쓰기 시 **Atomic Write(원자적 기록)** 로직을 적용하여 비정상 종료 시에도 데이터 손실을 원천 차단.

### 나. 무한 피라미드 종속 보상 시스템 (Line 302, 310, 318 준수)
- **재귀적 보상 전파:** 신규 유저(C) 가입 시 -> 엔진이 추천인(B)의 지갑 주소 파일 로드 및 보상 갱신 -> 추천인(B)의 추천인(A) 지갑 주소 파일 로드 및 연쇄 갱신.
- **즉시 반영:** 채굴 중이라도 부모 노드의 개별 파일을 즉시 업데이트하여, 부모가 로그인/새로고침 시 실시간으로 늘어난 보상 수치를 확인 가능하게 함.
