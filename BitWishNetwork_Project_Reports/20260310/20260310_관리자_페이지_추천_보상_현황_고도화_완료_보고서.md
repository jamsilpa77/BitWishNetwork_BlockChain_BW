## 1. 주요 변경 및 구현 사항

### **(1) 가입자 목록 테이블 컬럼 확장 (UI)**
- **보너스 지급 상태:** 모든 가입자에게 지급된 2% 보너스 상태를 시각화.
    - **표출:** `지급완료 (2 %)`
    - **스타일:** **보라색 (Purple, #a855f7)** 적용하여 가치 강조.
- **KYC 상태:** 향후 시스템 연동을 대비한 전용 라벨링 시스템 구축.
    - **표출:** `미신청` (기본값)
    - **스타일:** 지정된 상태별 컬러 맵핑 완료 (빨간색, 파란색, 녹색 등).

### **(2) 실시간 리스트 요약 헤더 (Header Integration)**
- **위치:** "👤 내 코드로 가입된 가입자 목록" 타이틀 우측.
- **항목:**
    - **전체 보상 ( ? BW):** 해당 추천인이 획득한 총 보상합 실시간 표출.
    - **전체 보너스 ( ? %):** 영입된 가입자들에게 정상 주입된 보너스율 명시.

### **(3) 백엔드 데이터 정밀 연동 (Logic)**
- `/api/admin/rewards/detail` API를 수정하여, 가입 기록만 보여주던 방식에서 벗어나 **실제 MiningState의 보너스율**을 1:1로 대조하여 반환하도록 고도화함.
