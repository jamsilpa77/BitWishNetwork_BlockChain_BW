# 🔌 BitWish Network 크롬 확장프로그램 초간단 설치 및 연동 가이드

> [!NOTE]
> **컴퓨터를 잘 모르시는 분들을 위해** 마우스 클릭 위주로 아주 쉽게 작성된 가이드입니다. 
> 개발자 용어 대신 누구나 따라 할 수 있는 단계별 순서로 설명해 드립니다.

---

## 💡 1. 실제 배포하면 유저가 홈페이지에서 클릭했을 때 팝업이 뜨나요? (가장 중요한 질문에 대한 답변)

**네, 완벽하게 잘 뜹니다!** 다만 아래와 같은 순서로 작동하게 됩니다.

1. **유저가 홈페이지에서 [Chrome Web Store 설치하기] 버튼을 클릭합니다.**
2. **구글 공식 크롬 웹스토어 페이지로 이동합니다.**
   * (저희가 개발한 이 확장프로그램 폴더를 구글 크롬 개발자 센터에 등록하고 심사를 받아 올려두어야 이 페이지가 열립니다.)
3. **페이지에 우리 확장프로그램 이름과 아이콘이 나타납니다.**
   * 우리가 폴더 안에 넣어둔 `manifest` 파일에 프로그램의 이름(`BitWish Network Mining Extension`)과 로고 이미지, 설명이 미리 기록되어 있어 구글 스토어에 자동으로 나타나게 됩니다.
4. **유저가 파란색 [Chrome에 추가] 버튼을 클릭해 설치합니다.**
5. **설치가 완료되면 브라우저 우측 상단(주소창 옆)에 자그만 BitWish 로고 아이콘이 추가됩니다.**
6. **유저가 그 아이콘을 누르면, 우리가 만든 예쁜 보너스 팝업 창이 눈앞에 나타나 정상 작동합니다!**
   * 홈페이지에서 단순히 설치하기 버튼을 누른다고 화면 중간에 바로 팝업이 뜨는 것이 아니라, **크롬에 설치 후 오른쪽 위 아이콘을 누르면 팝업이 열리는 방식**입니다.

## 🛠️ 2. 크롬 확장프로그램 수동 설치 및 테스트 방법 (마우스만 따라 하세요!)

개발 중에는 크롬 웹스토어에 등록하기 전에 브라우저에 임시로 넣어서 테스트를 해야 합니다. 보내주신 탐색기 화면을 기준으로 아주 쉽게 설명해 드릴게요.

### 1단계: 설치할 폴더 준비하기
* 보내주신 캡처 화면 속의 폴더(`BitWishNetwork_ChromeExtension`)에 들어있는 파일(background, content 등)들은 **직접 더블클릭해서 실행하는 파일이 아닙니다.**
* 크롬 브라우저가 이 폴더를 통째로 읽어가서 실행하도록 세팅해야 합니다.

### 2단계: 크롬 브라우저 설정창 열기
1. 인터넷 창(**Chrome 브라우저**)을 켭니다.
2. 주소 입력창에 아래 주소를 그대로 복사해서 넣고 엔터(Enter)를 칩니다.
   ```text
   chrome://extensions/
   ```
3. 창이 열리면 화면 **오른쪽 맨 위**에 있는 **[개발자 모드(Developer mode)]**의 토글 스위치를 마우스로 클릭하여 **켭니다 (파란색으로 활성화)**.

### 3단계: 폴더 통째로 등록하기
1. 개발자 모드를 켜면 화면 **왼쪽 맨 위**에 **[압축해제된 확장 프로그램을 로드합니다]** 라는 버튼이 나타납니다. 이 버튼을 클릭합니다.
2. 폴더 선택 창이 뜨면, 보내주신 캡처 화면에 나온 폴더인 `c:\BitWishNetwork_BlockChainMainnet\BitWishNetwork_ChromeExtension` 폴더를 클릭하고 아래의 **[폴더 선택]** 버튼을 누릅니다.
   * *주의: 폴더 안으로 더블클릭해서 들어간 뒤 파일을 고르는 것이 아니라, `BitWishNetwork_ChromeExtension` 폴더 자체를 1번 클릭하고 아래 [폴더 선택] 버튼을 누르는 것입니다.*
3. 목록에 **"BitWish Network Mining Extension"** 카드가 새로 생기면 수동 설치 성공입니다!

### 4단계: 브라우저 상단에 아이콘 고정하기
1. 크롬 브라우저 우측 상단(주소창 옆)에 있는 **퍼즐 조각 모양 아이콘(🧩)**을 클릭합니다.
2. 목록에서 **BitWish Network Mining Extension**을 찾은 뒤, 옆에 있는 **핀 모양 아이콘(📌)**을 클릭해 파랗게 만듭니다.
3. 이제 주소창 옆에 노란색 BitWish 아이콘이 상시 노출됩니다.

## 🧪 3. 채굴 보너스 30% 실시간 작동 테스트하기

설치가 완료되었으면, 제대로 채굴 홈페이지와 연동되어 보너스 30%가 가산되는지 직접 눈으로 검증할 수 있습니다.

1. **홈페이지 실행 및 로그인**
   * 홈페이지(`http://localhost:5000`)에 접속하여 지갑 주소로 로그인하고, 마이닝(채굴)을 시작합니다.
2. **확장프로그램 팝업 열기**
   * 크롬 주소창 옆에 고정된 **BitWish 노란색 아이콘**을 클릭하면, 보너스 연동 팝업창이 열립니다.
3. **지갑 연결하기**
   * 팝업창에 홈페이지에서 사용 중인 본인의 **지갑 주소**를 마우스로 복사해서 붙여넣은 뒤, **[지갑 연결 / 연동하기]** 버튼을 클릭합니다.
   * 연동이 성공하면 현재 실시간 채굴 속도와 보유 포인트가 팝업 창에 그대로 나타납니다.
4. **보너스 30% 인증 테스트**
   * 팝업창의 니모닉 보안 인증 칸에 본인의 니모닉 구절을 입력하고 **[니모닉 보안 인증]** 버튼을 클릭합니다.
   * 완료 메시지가 뜨며, 백엔드 데이터베이스에 즉시 반영되어 채굴 보너스가 **기존 10%에서 30%로 급증**합니다.
   * 마이닝 홈페이지 화면상의 실시간 채굴 속도(Rate)가 30% 만큼 실시간으로 증가하여 가운팅되는지 확인합니다.
5. **출석 체크 5% 추가 보너스 테스트**
   * 팝업창의 **[출석 체크 및 5% 부스트 받기]** 버튼을 누르면 당일 출석 보너스 5%가 실시간으로 추가 적용됩니다.
