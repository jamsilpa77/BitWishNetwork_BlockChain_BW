## 🎁 **보너스 시스템 상세 구조**

### **1. 출석 보너스 시스템 (5%)**

#### **출석 보너스 체계**
- **보너스율**: 5% (0.05) - 출석 체크 완료 시
- **보너스율**: 0% (0.00) - 출석 체크 미완료 시
- **일일 보상 증가**: 0.2625 BW/시간 (출석 시)

#### **출석 체크 시스템**
- **출석 가능 시간**: 정각 오전 9시 ~ 다음날 오전 8시 59분 59초
- **달력 날짜 표시**:
  - **활성화**: 파란색 (출석 가능)
  - **비활성화**: 회색 (출석 기회 상실, 영원히 유지)
  - **출석 완료**: 빨간색 (출석 체크 완료, 영원히 유지)

#### **출석 보너스 계산 공식**
```
출석_보너스율 = 출석_체크_완료시_5% (0.05)
최종_채굴률 = 기본_채굴률 × (1 + 출석_보너스율)
```

#### **구체적 계산 예시**
- **출석 체크 완료 사용자**:
  - 기본 채굴률: 0.25 BW/시간
  - 출석 보너스: 5% (0.05)
  - 최종 채굴률: 0.25 × (1 + 0.05) = **0.2625 BW/시간**
  - 일일 추가 수익: 0.0125 × 24 = **0.3 BW/일**
  - 월간 추가 수익: 0.3 × 30 = **9 BW/월**

- **출석 체크 미완료 사용자**:
  - 기본 채굴률: 0.25 BW/시간
  - 출석 보너스: 0% (0.00)
  - 최종 채굴률: 0.25 × (1 + 0.00) = **0.25 BW/시간**

### **2. 추천 보너스 시스템 (2% + 1BW)**

#### **추천인 보너스 구조**
- **추천인**: 2% 영구 적용 (가입자 발생 시마다 추가 반영)
- **가입자**: 1회만 2% 영구 적용 (재적용 불가)
- **즉시 보너스**: 추천인과 가입자 각 1BW 즉시 지급
- **영구 관계**: 추천인 관계는 한 번 설정되면 영구적으로 유지

#### **추천 보너스 보관함 시스템**
- **추천 보너스 보관함**: 2% 보너스 별도 보관
  - 계산: 기본 보상률 0.25BW × 1명당 2% = 별도 보관
  - 가입자 추가 시 지속적으로 누적 보관
  - KYC 통과 후 본인 "실시간 누적 보상"에 반영

- **추천 보상 보관함**: 1BW 보상 별도 보관
  - 추천인과 가입자 각각 1BW 보관
  - KYC 통과 후 "실시간 누적 보상"에 반영

#### **추천 보너스 페이지 구조**
- **가입자 목록**: 가입자별 지갑주소, 가입일시, 실시간 누적 보너스 채굴 수량
